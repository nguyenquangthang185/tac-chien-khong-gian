Chương trình của bạn về bản chất là một mô phỏng tác chiến 2D bằng OpenGL/FreeGLUT.
Nó không phải game engine hoàn chỉnh, mà là một chương trình đồ họa thời gian thực gồm:
  - phần dữ liệu trạng thái
  - phần cập nhật logic
  - phần vẽ giao diện và đối tượng
  - phần xử lý input
  - vòng lặp chạy liên tục
